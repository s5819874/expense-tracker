## Getting Started

用Node.js + Express框架與MongoDB資料庫建立的記帳清單。

#### Home Page

![Webpicture](https://github.com/s5819874/expense-tracker/blob/master/homepage.png)

## Features
* 瀏覽所有消費記錄與合計金額
* 新增, 編輯與刪除消費記錄
* 依照類別進行篩選，顯示該類別的消費紀錄與合計金額
* 簡易分析各類別的消費總額


## Environment Setup
* Node.js

## Built With & Tools
* Express framework
* Express-handlebars
* body-parser
* mongoose
* method-override
* CDN include Bootstrap, jQuery, Popper.js and fontawesome
